## Part D: Conclusion logic

```python
def compute_verdict(checks: list[tuple[bool, str]]) -> str:
    """GO_FOR_P3_TRAINING if all 9 checks PASS (SKIP counts as PASS with documentation).
    FIX_FIRST if any check FAILS."""
    failed = [name for passed, name in checks if not passed]
    if failed:
        return "FIX_FIRST"
    return "GO_FOR_P3_TRAINING"
```
